# 🚀 Quick Start: Push to GitHub

## Three Simple Steps

### 1️⃣ Run the Setup Script
```powershell
.\Setup-GitHubRemote.ps1
```

### 2️⃣ Follow the Prompts
The script will guide you through:
- Authentication check
- Repository creation
- Pushing your code

### 3️⃣ Done! ✅
Your repository will be live on GitHub!

---

## Alternative: Manual Method

```powershell
# 1. Create repo on https://github.com/new
#    Name: Learning_AdvancedCSharpStudies
#    Don't initialize with any files

# 2. Add remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/Learning_AdvancedCSharpStudies.git

# 3. Push
git push -u origin main
```
